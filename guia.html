ETIQUETAS

ESTRUCTURA BÁSICA
  • html: abrir documento
  • head: info acerca del documento como enlaces, hojas de estilo, etc...
  • body: cuerpo del documento
DEFINIR ELEMENTOS EN UNA PÁGINA
  • title: título de la página o documento
  • div: sección en una pág o doc
  • img src: insertar imágenes
  • link: relación entreuna pág o doc con un elem externo
  • style: definir el estilo de la info de una pág o doc. Se puede escribir CSS directamente en el head
ENCABEZADO Y PÁRRAFO
  • h1, h2, h3, h4... h6: encabezados o títulos
  • p: definir un párrafo
  • br: salto de línea
  • b: resaltar texto en negrita
  • i: texto en cursiva
ENLACES
  • a: hipervínculos
LISTAS
  • li: definir elemento u objeto dentro de una lista
  • ol: definir una lista ordenada de elementos
  • ul: definir lista desordenada
TABLAS
  • table: definir tabla
  • tr: filas
  • td: representar celda en una tabla
ATRIBUTOS
    ATRIBUTOS BÁSICOS
  • class: normalmente se utiliza con CSS para aplicar estilos a los elem que tienen prop en común
  • id: indicador único para cada elemento para que se pueda acceder a él desde CSS o JV
  • style: aplica los estilos CSS a un elem anulando los estilos anteriores
  • title: títulos de los elem. El título de un elem se mostrará cuando el cursor esté sobre él (mejor accesibilidad
    • ATRIBUTOS DE INTERNACIONALIZACIÓN
  • lang: idioma o lenguaje del elem
  • dir: dirección del texto. ltr izq-der. rtl der-izq
    • ATRIBUTOS DE FOCO
  • acceskey: establece una tecla de acceso rápido que permite activar, o poner el foco, en un elem

  • tabindex: sobreescribe el orden de tabulación que tiene predeterminado el navegador y usa el especificado. Valor entre 0 y 32.767. Ej: Importante definirlo para determinar el orden de navegación de los campos que componen un formulario.
    • ATRIBUTOS DE EVENTOS
      EVENTOS DE RATÓN
    • onclick: ejecuta la acción cuando se hace click sobre el elem
    • lang: idioma o lenguaje del elem
